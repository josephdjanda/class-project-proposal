Application Overview

A user will be able to view a website that has a wide range of information about different aircraft in the United States Military. They will be able to view pictures of many different types of aircraft. When they click on the picture of the aircraft it will take them to an attached page that will have information about the aircraft.

Technology Overview

Languages that are used will be HTML and Css and the website will be publically hosted using Github
